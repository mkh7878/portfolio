Resume Chat Bot


This chatbot is a Flask web application designed to interact with users by answering questions based on the content of a provided resume (mine). It uses natural language processing through the LangChain framework integrated with OpenAI’s language models. The chatbot is tailored to analyze and interpret the resume’s content, providing contextually relevant responses to user questions.

The chatbot uses a customizable prompt template within LangChain to structure its responses. This template ensures that the generated text is coherent, contextually appropriate, and adheres to specific instructions, such as providing detailed explanations or encouraging further questions.
